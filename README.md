Simple Python Calculator

A basic Python calculator program that performs simple arithmetic operations such as addition, subtraction, multiplication, and division.

Features
Addition
Subtraction
Multiplication
Division
Handles division by zero errors

Requirements
Python 3.x

How to Run
Save the code in a file named calculator.py
Open your terminal or command prompt

Run the program using:
python calculator.py

Sample Output
1. Addition
2. Subtraction
3. Multiplication
4. Division
Choose an operation: 1
Enter first number: 5
Enter second number: 3
Result: 8
The result is: 8
Code Overview

The program:

Displays a menu of operations
Accepts user input for the operation
Takes two numbers as input
Performs the selected calculation
Displays the result
Notes
Division by zero is not allowed and will display an error message.
The program currently accepts integer inputs only.
